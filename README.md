# CORE.AI - Premium AI SaaS Feature Page

CORE.AI is a high-end, motion-driven marketing page for an AI-powered agency intelligence platform. It features premium animations, 3D scroll-driven experiences, and a sleek, dark-themed UI designed to showcase next-generation AI capabilities.

## 🚀 Experience the Motion
This project leverages advanced animation techniques to create an immersive user journey:
- **GSAP ScrollTrigger**: Powering the entire scroll-driven experience, including the 3D card flips in the Insights section.
- **Lenis Smooth Scroll**: For a luxurious, buttery-smooth scrolling experience.
- **Framer Motion**: Used for micro-interactions and smooth component entrances.
- **3D Perspective**: Interactive sections that shift and rotate based on your scroll progress.

## ✨ Key Features
- **3D AI Capabilities**: A deep-dive 3D panel that reveals the core intelligence dashboard, analytics, and coaching tools.
- **Interactive Eye Reveal**: A stunning visual section that transforms as you scroll.
- **Dynamic Leaderboards**: Visual representation of performance and team branch metrics.
- **Responsive Staging**: Fully optimized for desktop, tablet, and mobile devices with fluid typography.

## 🛠️ Tech Stack
- **Framework**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4.0](https://tailwindcss.com/)
- **Animation**: [GSAP](https://greensock.com/gsap/) (ScrollTrigger, @gsap/react), [Motion](https://motion.dev/)
- **Typescript**: For robust, type-safe development.
- **Icons**: [Lucide React](https://lucide.dev/)

## 🏁 Getting Started

### Prerequisites
- Node.js (Latest LTS version recommended)

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the development server:
```bash
npm run dev
```
The app will be available at `http://localhost:3000`.

### Building for Production
Create an optimized production build:
```bash
npm run build
```
